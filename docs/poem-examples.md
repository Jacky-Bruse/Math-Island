# 古诗示例

这个文档提供两类示例：

- 手动新增古诗时可直接参考的字段示例
- 批量导入 Markdown 文件时可直接复用的文本示例

## 手动新增示例

### 示例 1：静夜思

- 标题：静夜思
- 朝代：唐
- 作者：李白
- 正文：

```text
床前明月光
疑是地上霜
举头望明月
低头思故乡
```

### 示例 2：春晓

- 标题：春晓
- 朝代：唐
- 作者：孟浩然
- 正文：

```text
春眠不觉晓
处处闻啼鸟
夜来风雨声
花落知多少
```

### 示例 3：登鹳雀楼

- 标题：登鹳雀楼
- 朝代：唐
- 作者：王之涣
- 正文：

```text
白日依山尽
黄河入海流
欲穷千里目
更上一层楼
```

## 带朝代和作者的录入示例

如果你希望古诗在页面里完整显示“朝代 + 作者”，建议按手动新增或编辑的方式录入。

### 示例 4：相思

- 标题：相思
- 朝代：唐
- 作者：王维
- 正文：

```text
红豆生南国
春来发几枝
愿君多采撷
此物最相思
```

### 示例 5：游子吟

- 标题：游子吟
- 朝代：唐
- 作者：孟郊
- 正文：

```text
慈母手中线
游子身上衣
临行密密缝
意恐迟迟归
谁言寸草心
报得三春晖
```

## Markdown 导入格式

项目当前的 Markdown 导入格式很简单：

- 每首古诗的第 1 行是标题
- 后面的非空行都视为正文
- 多首古诗之间用单独一行 `---` 分隔
- 当前导入只解析标题和正文，不解析作者、朝代

也就是说，下面这种“带朝代和作者”的写法目前不能直接按结构识别：

```text
静夜思
唐 · 李白
床前明月光
疑是地上霜
举头望明月
低头思故乡
```

如果你这样导入，`唐 · 李白` 会被当成正文的第一行，而不是元数据。

## Markdown 导入示例

下面这个文件可以直接作为导入示例使用：

```markdown
静夜思
床前明月光
疑是地上霜
举头望明月
低头思故乡

---

春晓
春眠不觉晓
处处闻啼鸟
夜来风雨声
花落知多少

---

江雪
千山鸟飞绝
万径人踪灭
孤舟蓑笠翁
独钓寒江雪
```

## 使用建议

- 需要保留作者和朝代时，优先用页面里的“手动新增/编辑”
- 需要一次导入很多首古诗时，优先用 Markdown 简化格式
- 导入前建议先检查标题是否重复，避免被识别为重复内容后跳过
